# Azure Managed Kubernetes Service Bug

This example reproduces the bug identified in this issue: https://github.com/Azure/azure-sdk-for-go/issues/2085

## Running this

The following Environment Variables need to be set:

 - `ARM_CLIENT_ID`
 - `ARM_CLIENT_SECRET`
 - `ARM_SUBSCRIPTION_ID`
 - `ARM_TENANT_ID`
